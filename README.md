# Java Exercises ☕

Este repositório contém uma coleção de exercícios em **Java** com foco no desenvolvimento de lógica de programação, estruturas de dados e conceitos fundamentais da linguagem.

O objetivo é praticar e consolidar conhecimentos importantes, além de fortalecer habilidades de resolução de problemas.

## 📚 Conteúdo

Os exercícios abordam temas como:

- Fundamentos da linguagem Java
- Estruturas de controle (if, switch, loops)
- Manipulação de Strings
- Arrays e Collections
- Programação Orientada a Objetos (POO)
- Estruturas de dados (Stack, List, Map)

## 🧠 Objetivo

- Melhorar a lógica de programação
- Praticar conceitos fundamentais de Java
- Construir um portfólio de exercícios resolvidos

## 🛠 Tecnologias

- Java
- Git e GitHub
